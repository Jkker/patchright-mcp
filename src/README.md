# Where is the source?

Patchright MCP source code is based on the [Playwright MCP](https://github.com/microsoft/playwright-mcp) from the [Playwright monorepo](https://github.com/microsoft/playwright/blob/main/packages/playwright/src/mcp), but uses [Patchright](https://github.com/Kaliiiiiiiiii-Vinyzu/patchright-nodejs) for undetected browser automation. Please refer to the contributor's guide in [CONTRIBUTING.md](../CONTRIBUTING.md) for more details.
